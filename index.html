Project 6

This project includes all the steps to buying a product in walmart.com.br. They are: Choosing the delivery address, Delivery Type, Delivery Schedule, Payment and Checkout. My tasks was prototyping in HTML5 and CSS3, as well as definitions of some jquery plugins to be used. The layout was created by the UX area of Walmart but with the short-term and greater responsibility for the development, also participated in suggestions for changes to the layout to better adaptation the desired responsiveness. This project is fully responsive. I used Photoshop, HTML5, SASS/COMPASS, Susy Grid, Bootstrap components, Grunt and Git (Bitbucket).

  • Client: Walmart
  • Date: October 2013
  • Service: Web Development
